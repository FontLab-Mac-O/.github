## 🖌 Overview  
FontLab Advanced is a professional font design and editing application used by type designers and typographers worldwide to create, edit, and master complete typefaces from initial concept through production-ready font file output.

The glyph drawing environment provides a complete vector path editor optimized specifically for letterform design, with contour tools, curve handles, and constraint systems calibrated for the precise curves and strokes that quality typeface design demands.

Spacing and kerning tools manage the complex task of establishing consistent sidebearings and pair-specific kerning values across a complete character set, with visual feedback and metrics windows that make the spacing process both precise and efficient.

OpenType feature programming lets you build ligatures, alternates, contextual substitutions, and every other OpenType capability directly into your font, controlling typographic behavior in supporting applications with standard OpenType syntax.

For type designers, lettering artists, and graphic designers who want to create custom typefaces at professional production quality, FontLab Advanced is the font design platform that covers every stage of the type design and production workflow.
